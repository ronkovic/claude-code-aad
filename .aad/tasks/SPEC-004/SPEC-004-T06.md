# SPEC-004-T06: orchestrateコマンド実装

## 複雑度: M

## 実装内容

`aad orchestrate`コマンドでオーケストレーターを起動し、登録されたSpecを実行するCLIコマンドを実装します。

- `cli/src/commands/orchestrate.rs`の実装
- `--specs`オプションで複数Spec指定
- 実行ログのコンソール表示
- 結果サマリーの表示
- `Orchestrator::on_session_completed()`メソッド実装
- `Orchestrator::on_session_failed()`メソッド実装
