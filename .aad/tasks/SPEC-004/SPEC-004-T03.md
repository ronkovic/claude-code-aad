# SPEC-004-T03: セッション登録・起動ロジック実装

## 複雑度: M

## 実装内容

Specをセッションとして登録し、依存関係に基づいて適切な順序で起動するロジックを実装します。

- `Orchestrator::register_spec(spec_id)`メソッド実装
- `Orchestrator::start_session(session_id)`メソッド実装
- 依存関係グラフに基づいた実行順序の決定
- 並列実行可能なSpecの同時起動
- セッション起動エラーハンドリング
