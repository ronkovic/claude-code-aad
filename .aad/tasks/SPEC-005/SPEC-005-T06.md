# SPEC-005-T06: BackupAdapter実装

## 複雑度: M

## 実装内容

ファイルバックアップ機能を実装します。

- `BackupAdapter` 構造体実装
- `backup(file_path)` メソッド実装
- `cleanup_old_backups(keep_count)` メソッド実装
- ISO 8601形式タイムスタンプ付きファイル名生成
- `.aad/backups/` ディレクトリ管理

## 変更ファイル

### 新規作成
- `crates/infrastructure/src/persistence/backup_adapter.rs`

### 変更

## テストケース

- [ ] `backup()`でファイルがバックアップされる
- [ ] バックアップファイル名にISO 8601形式のタイムスタンプが含まれる
- [ ] バックアップファイルが`.aad/backups/`に保存される
- [ ] `cleanup_old_backups(10)`で10世代より古いバックアップが削除される
- [ ] バックアップ対象ファイルが存在しない場合はエラーを返す
