## 概要

Rustワークスペースの基本構造を作成し、`domain`クレートの骨格を実装します。

## 作業内容

### 1. ルートCargo.toml作成
- ワークスペース設定を定義
- 共通の依存関係を設定

### 2. crates/domain/構造作成
- `crates/domain/Cargo.toml`を作成
- `crates/domain/src/lib.rs`を作成
- 基本的なモジュール構造を定義

### 3. 基本モジュール構成
```
crates/domain/src/
├── lib.rs
├── entities/
│   └── mod.rs
├── value_objects/
│   └── mod.rs
├── repositories/
│   └── mod.rs
└── error.rs
```

## 変更ファイル

- `Cargo.toml`（新規）
- `crates/domain/Cargo.toml`（新規）
- `crates/domain/src/lib.rs`（新規）

## 受け入れ基準

- [ ] `cargo build` がエラーなく実行できる

