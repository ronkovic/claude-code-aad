## 🔗 API仕様

### エンドポイント1: ユーザー登録

**リクエスト**
```http
POST /api/users
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "securepassword"
}
```

**レスポンス（成功）**
```http
201 Created
Content-Type: application/json

{
  "id": "uuid",
  "email": "user@example.com",
  "created_at": "2026-01-11T10:00:00Z"
}
```

**レスポンス（エラー）**
```http
400 Bad Request
Content-Type: application/json

{
  "error": "Email already exists"
}
```
